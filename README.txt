DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/

Developed with Python 3.2.2 and PyGame 1.9.2 pre on Lubuntu.

RUNNING THE GAME:

Open a terminal / console and "cd" to the game directory and run:

  python run_game.py

HOW TO PLAY THE GAME:

Gameplay:
  
Controls:
  SPACEBAR - Start the Game
  F11 - During gameplay sets the game full screen
  w - forward
  s - back
  a - turn left
  d - turn right
  m - display/hide map
  n - collect a component
  j - fire blaster
